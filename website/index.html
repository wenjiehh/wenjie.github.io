Intro

201406-201809
我於2014年進入清大經濟系就讀，2018年畢業班排名為1/54（成績單），獲得三個學期書卷獎。大四獲得教育部學海飛颺留學獎學金，至日本大阪大學當一學年的交換生。

201809-202202
從日本回來後即畢業進入職場，在一外商公司工作，連續兩年獲得公司獎項分別為Achievement Prize、Team Award。 在職時利用下班後時間進修，參與台大資訊系統訓練班開設的Python課程並取得結業證書、及參訓政府設立之產業人才投資計畫，課程包含：物聯網技術[利用arduino、ESP32開發版收集到的溫/濕度資料上傳雲端ThingSpeak]、前端網頁設計[編寫html, CSS, JavaScript]、SQL等等，並通過考核結訓。

原先是為解決工作上的問題而開始學習程式，而後像開啟了一扇窗，對世界的看法有了完全不同以往的角度！亦體驗到學習新知識帶來的愉悅。後決定考取資工所，希望深入研究相關領域。

202202-202206
為達到目標，我先以台大國企所碩一的身分在台大就讀，並修讀資工相關課程：C/C++程式設計、商管程式設計（Python）、作業研究等。透過課程中的作業、projects（詳細請參閱Works），持續累積實作能力。

202206-202302
準備資工所考試，並利用各大學開放式課程學習:計算機結構（黃婷婷教授）、離散數學（韓永楷教授）、資料結構（韓永楷教授）、演算法（江蕙如教授）、作業系統（周志遠教授）、Algorithm Design and Analysis（陳縕儂教授）、線性代數（莊重教授）等課程。

202303-202306
選修台大課程：高等演算法、資料結構與演算法、人工智慧導論、數位系統與實驗、機率；並旁聽電子設計自動化導論、電路學(因外系身分無法選修，但徵得授課教師同意旁聽)。

Works

  • Programming for Business Computing-Final Project (Team Project)
    台大課程──商管程式設計Final Project，課程成績A。內容為：爬取使用者在NTU COOL（課務系統）未逾期的作業及考試，並按截止日期列出、在逾期之前跳出提醒。本專案希望藉由提醒即將到期的作業和考試，避免忙碌的台大學生忘東忘西、遲交作業。
    我負責撰寫提醒視窗的程式（影片00:40處），利用Tkinter函式庫建出視窗畫面，透過引數傳入截止日將至的事件以表格呈現於視窗。視窗會依照傳入的任務數量決定視窗建立的大小，以避免單一固定大小造成畫面太空曠或太擁擠。炸彈圖示搭配秒針急促行走的背景音效，希望給用戶達到緊張的效果，藉以盡快執行未完成的任務。

  • Operations Research-Final Project (Team Project)
    台大課程──作業研究Final Project，利用OR習得的技巧，及Python的Integer Program solver、pandas資料視覺化，應用於國企之夜排班法則最佳化。
    Input：每位同學可排班的時段、對於不同工作的志願序
    Output：每位同學在每一個時段應該做或不做哪一個工作
    Objective：最小化人力空缺、最小化同學所執行工作的工作志願
  • C/C++ Programming
    台大課程──C/C++程式設計，課程成績A，Judge System for NTU CS Training題目AC數為同期排名第2。持續進修C，準備考取CPE檢定。
  • Python Programming
    利用Python套件BeautifulSoup爬蟲抓取財政部稅務網站上的統一發票中獎號碼，透過字串切片及條件判斷中獎與否、中獎金額。使用者可選擇對獎月份，並選擇是否繼續對獎，最後結束計算加總中獎金額。
